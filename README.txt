audio_ALC_guides
============
OS X Realtek ALC AppleHDA Onboard Audio Guides

Guides describe Realtek ALC AppleHDA onboard audio requirements, installation methods, custom capabilities and problem solving.

Problem Reporting, see https://github.com/toleda/audio_realtekALC Problem Reporting

Requirements
  A. OS X
	1. Yosemite/10.10 or newer
	2. Mavericks/10.9 or newer
	3. Mountain Lion/10.8 or newer
  B. Native AppleHDA.kext  (If not installed, run OS X Installer)
  C. Codec/ALC Support (auto detection)
	1. 269 (BRIX only)
	2. 283 (BRIX Pro and NUC)
	3. 885
	4. 887
	5. 888
	6. 889
	7. 892
	8. 898
	9. 1150
  D. Audio ID Support (Definitions, Note A)
	1. 269, 283. 885, 887, 888, 889, 892, 898, 1150
	2. 887, 888, 889, 892, 898, 1150
	3. 887, 888, 889, 892, 898

Notes
  A. Audio ID Definitions (Layout/Audio ID injection installed separately, 
        see https://github.com/toleda/audio_ALCInjection)
	1 - 3/5/6 audio port analog audio
	2 - 3 audio port analog 5.1 surround sound
	3 - HD3000/HD4000 HDMI audio with analog audio
